# GO enrichment scripts and plots

Revigo_C*_HDBSCAN.R are the R script written to plot biological processes, starting from Revigo's output.
  If you want to replicate this kind of plot: 
  1. download the scatterplot Revigo R script
  2. copy and paste the "revigo.data" object where indicated in the "to_replicate.R" script, where indicated.
  3. modify the minPts variable where indicated, it set the minimum number of dots considered to define a cluster.
  4. modify color where indicated.
